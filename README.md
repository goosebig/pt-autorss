# pt-autorss

Auto fetch and download torrents even can auto load torrents.

## Usage
```bash
npm i --production
cp config-example.js config.js
vim config.js
node . # or npm start
```

To config pt-autorss, you would learn a little about Javascript.

More custom rule see `lib/NexusPHP-utils.js`.

PR welcome.
